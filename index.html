PyUSB - Easy USB access on Python

PyUSB aims to be an easy to use Python module to access USB devices. PyUSB relies on a native system library for USB access. Currently, it works out of the box with libusb 0.1, libusb 1.0, libusbx, libusb-win32 and OpenUSB. Python 3 is the preferred runtime since the EOL of Python 2.7, but most things should still work with Python 2 versions starting at 2.4.

Documentation

The best way to start with PyUSB is through the PyUSB tutorial, and API reference can be found in the doc strings.

Support

If you need support, you can subscribe to the mailing list. At the SourceForge page you can find instructions on how to subscribe.

Development

PyUSB development happens at github, and signed dist tar ball releases are available.
